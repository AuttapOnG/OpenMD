# OMD-012: Math rendering (KaTeX) + footnotes

**Status:** in_progress (2026-07-10)
**Spec:** docs/specs/OMD-012-katex-footnotes.md (approved in chat 2026-07-10)
**Plan:** docs/plans/2026-07-10-omd-012-katex-footnotes.md

## Approach
- Server-side KaTeX: markdown-it-texmath (delimiters: 'dollars',
  throwOnError: false) + markdown-it-footnote in createMarkdownParser().
- Browser gets only katex.min.css + 20 woff2 fonts (~320 KB), vendored by
  esbuild into media/katex/ preserving the katex/fonts/ layout so the css's
  relative url(fonts/...) resolves on all three surfaces (served /assets/,
  webview asWebviewUri, staged .temp/assets).
- Behaviors verified on host before planning: "$5 กับ $10" stays literal,
  \$ escape works, invalid TeX doesn't throw, $$ gets katex-display.
- katex 0.17.0 ships its own types; texmath/footnote declared in types.d.ts.

## Execution
Delegated to Codex CLI (codex exec --full-auto, background) per the
delegation workflow: deps preinstalled on host (Codex has no network),
Codex skips server tests (can't bind ports), host re-verifies everything
and owns commits/bookkeeping.
