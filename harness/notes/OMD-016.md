## How the GIF was recorded (repeatable, no human screen recording)
`PreviewServer` + `render.ts` have no vscode imports, so:
1. `scratchpad/run-server.js` — node script requiring `out/server.js` +
   `out/render.js`, assetsDir=`media/`, registers a demo md, prints URL.
2. Playwright MCP at 1100x700, device-scale screenshots per scene;
   auto-refresh scene = edit the md while the page is open, wait >1s poll.
3. Pillow assembles PNGs → GIF (256-color adaptive palette, per-frame
   durations, `optimize=True`). Script pattern kept in session scratchpad;
   recreate freely — total ~10 min.
