## Notes

(Record decisions, surprises, and gotchas here while implementing.)

- Design decision (2026-07-08): minimal localhost HTTP server with 1s mtime
  polling. file:// pages cannot fetch (browser security), so pure static
  polling is impossible; meta-refresh rejected (constant reload flicker);
  WebSocket rejected (the v0.0.3 Live Server was reverted for this complexity).
- Once this ships, the v1.0.0 mirror-path temp files only serve as the assets
  home — HTML is rendered on demand by the server.
